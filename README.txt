1. key in "make" in Cygwin, each algorithm will be compiled and put exe in the bin folder
2. run write_input to create shell script execute, meanwhile input user-defined n_loop
3. key in "sed -i 's/\r//g' execute" in Cygwin to fix \r behind each input's filename 
4. key in "sh execute" in Cygwin to run six all to all shortest path algorithms,162 input files for each
5. the result of each algorithm will be put in the results folder with the filename my+algorithm.txt
